My developments

The Café Management System is a C-based program designed to streamline the billing process in a café. It allows users to select items from a user-friendly menu, place orders, and generate bills efficiently. This system improves operational efficiency by reducing processing times and providing accurate bill generation for customers.

The Library Management System is a C-based program that facilitates efficient management of library operations. It includes features for cataloging books, managing purchases, and generating automated bills and also allows searching of books through book details like author, genre and price.

Campus dmit is a Python and SQL-based student portal designed to streamline the college search and application process. It allows users to search for colleges based on criteria such as cutoff marks, domain, fees, and location.

The Weather and Resource Utilization Tracker, part of the GeoSence project, is designed to monitor climate and resource usage data. It features a `report` class that generates detailed weather and resource utilization reports based on stored data.
